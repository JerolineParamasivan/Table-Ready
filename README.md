# TableReady — Frontend Replica

This repository contains a minimal Flask scaffold and a replica of the provided landing page using HTML, CSS, Bootstrap and a small amount of JS.

Quick start:

1. Create a virtual environment and install dependencies:

```bash
python -m venv .venv
source .venv/Scripts/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the app:

```bash
set FLASK_APP=app.py   # Windows
flask run
```

Then open http://127.0.0.1:5000 in your browser.

Files added/updated:
- `app.py` — minimal Flask server
- `templates/index.html` — page markup
- `static/css/styles.css` — custom styles
- `static/js/script.js` — minimal behavior

Adjust content, images and copy as needed.
